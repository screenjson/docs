---
title: Analysis
description: Derived retrieval-oriented data attached to a canonical document.
---

The `analysis` block carries derived data for retrieval and AI workflows. Optional and discardable вАФ a document without `analysis` is still canonical.

## Shape

```json
{
  "analysis": {
    "embeddings": { "<target-uuid>": [ /* embeddings */ ] },
    "passages":   [ /* passages */ ],
    "summaries":  [ /* summaries */ ],
    "settings":   { /* how this was produced */ },
    "meta":       {}
  }
}
```

## `embeddings`

A map keyed by the **UUID of the target** (a scene, element, or character). Value is an array of one or more embedding records.

```json
{
  "id":         "<embedding-uuid>",
  "model":      "text-embedding-3-large",
  "dimensions": 1536,
  "values":     [0.023, -0.041, ...],
  "source":     "text",
  "lang":       "en",
  "tokens":     420,
  "created":    "2026-01-14T10:30:00Z"
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `id` | uuid | вЬУ | Embedding identifier. |
| `model` | string (1вАУ100) | вЬУ | Model identifier. |
| `dimensions` | integer 1вАУ4096 | вЬУ | Vector length. |
| `values` | number[] 1вАУ4096 | вЬУ | Vector payload. |
| `source` | enum | вЬУ | Which field was embedded. `name`, `text`, `desc`, `heading`, `composite`. |
| `created` | datetime | вЬУ | ISO 8601. |
| `lang` | lang | вАФ | BCP 47 tag the source text was drawn from. |
| `tokens` | integer вЙ•1 | вАФ | Token count of the source text. |
| `meta` | meta | вАФ | Extension. |

Keys in the `embeddings` map are matched against the UUID pattern вАФ only valid RFC 4122 strings are accepted.

## `passages`

Retrieval-sized text chunks keyed to source elements.

```json
{
  "id":       "<uuid>",
  "scene":    "<scene-uuid>",
  "elements": ["<element-uuid>", "<element-uuid>"],
  "text":     { "en": "..." },
  "tokens":   256,
  "overlap":  32
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `id` | uuid | вЬУ | |
| `scene` | uuid | вЬУ | |
| `elements` | uuid[] (вЙ•1) | вЬУ | Element ids covered by this passage. |
| `text` | [text](/specification/shared-types#text) | вЬУ | Passage text, language-keyed. |
| `tokens` | integer вЙ•1 | вЬУ | |
| `overlap` | integer вЙ•0 | вАФ | Sliding-window overlap in tokens. Default `0`. |
| `meta` | meta | вАФ | Extension. |

## `summaries`

Document- or scene-scoped summaries.

```json
{
  "id":        "<uuid>",
  "scope":     "scene",
  "target":    "<scene-uuid>",
  "generated": true,
  "model":     "claude-opus-4-7",
  "text":      { "en": "..." },
  "created":   "2026-01-14T10:45:00Z"
}
```

| Field | Type | Required | Description |
|---|---|---|---|
| `id` | uuid | вЬУ | |
| `scope` | enum | вЬУ | `document` or `scene`. |
| `text` | [text](/specification/shared-types#text) | вЬУ | Summary text. |
| `created` | datetime | вЬУ | ISO 8601. |
| `target` | uuid \| null | вАФ | Scene id when `scope="scene"`. |
| `generated` | boolean | вАФ | `true` for machine-generated. Default `false`. |
| `model` | string (вЙ§100) | вАФ | Producing model id. |
| `meta` | meta | вАФ | Extension. |

## `settings`

Records how derived data was produced вАФ for reproducibility.

| Field | Type | Default | Description |
|---|---|---|---|
| `model` | string (вЙ§100) | вАФ | Embedding or summary model. |
| `size` | integer вЙ•1 | `512` | Chunk size. |
| `overlap` | integer вЙ•0 | `64` | Chunk overlap. |
| `tokeniser` | string (вЙ§50) | вАФ | Tokeniser identifier вАФ e.g. `cl100k`. |
| `meta` | meta | `{}` | Extension. |
